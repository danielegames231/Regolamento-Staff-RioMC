# Regolamento Staff SMP Minecraft

Sito pronto per GitHub Pages.

## Come pubblicarlo
1. Vai su GitHub.
2. Crea una repository chiamata `regolamento-staff-smp`.
3. Carica questi file nella repository.
4. Vai su Settings > Pages.
5. In Source scegli `Deploy from a branch`.
6. Seleziona branch `main` e cartella `/root`.
7. Salva.

Il sito sarà online dopo pochi minuti.
